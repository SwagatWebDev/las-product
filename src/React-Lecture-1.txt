Let us understand the installation process of React JS.
----------------------------------------------------------
1. Download NodeJS. What is node js let us understand.
NodeJS ek javascript ka runtime hain jo apki computer main javascript execute karne main madat karta hain.
Once you install nodeJS then only you will able to run JavaScript without browser.
Ham chate hain ki bina browser ham javascript ko run karen isliye hame nodejs ko install karna padega.

2. Download an editor VS Code or IntelliJ Idea.

3. Check your node version by -> node --version
Check your NPM version by -> npm --version

4. Search React Dev tool then add it to the Chrome browser extension
Ye extension apke sare ke sare component ko easy load karta hain and visually show karte hain.
